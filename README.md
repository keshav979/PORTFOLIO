# 🚀 3D Portfolio Website

A futuristic **3D personal portfolio** built using:

- Three.js
- Glassmorphism UI
- Particle background
- Smooth animations
- Interactive 3D objects

## ✨ Features

✔ 3D animated background  
✔ Smooth scrolling  
✔ Glass UI cards  
✔ Hover tilt effects  
✔ Interactive projects section  
✔ Modern dark theme  

## 🛠 Technologies

- HTML
- CSS
- JavaScript
- Three.js

## 📦 Installation

Clone the repository

git clone https://github.com/yourusername/3d-portfolio.git

Open the folder

cd 3d-portfolio

Run using live server or open:

index.html
